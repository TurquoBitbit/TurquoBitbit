        * [愛樂壓](https://aeropress.com/)
            * 我的配方:
                1. 中深焙豆16克，磨到接近espresso的細度
                2. 愛樂壓採用倒置法放入咖啡粉
                3. 熱水沖濕濾紙，開始計時
                4. 注入85度C熱水100克，攪拌均勻
                5. 蓋上蓋子壓出空氣，浸泡到100秒
                6. 翻過來擠出咖啡，加鮮奶180克飲用<br /><br />
